Step 1 :

run 

```
npm install 
```
in the terminal to install all the dependencies

Step 2 :

Then run 

```
mongod
```
to start the mongodb server

Step 3 :

Then run 

```
nodemon server.js
```
to start the server at port 3000

step 4 :

visit the url to see the app - 

```
http://localhost:3000/
```
